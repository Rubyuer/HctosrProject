# 晴天体育

  "晴天体育"是一款线上预约足球学院下的课程和教练的应用。

  主要功能:   
  
    1.核心模块：足球学院、足球课程、足球教练；
      足球学院：提供十多家足球学院，每个学院下都有其相关课程；
      足球课程：青少年足球培训课程、少儿足球培训课程；
      足球教练：每个课程下配备数名专业教练。
    
    2.打卡：打卡用于展示用户签到数据
    
    3.我的订单：全部订单、待使用订单、已使用订单、已取消订单等。
